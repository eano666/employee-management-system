Employee Management System (EMS) is a full-stack Spring Boot project designed to manage employee records efficiently. 
It demonstrates CRUD operations, database integration with MySQL, and a responsive frontend using Thymeleaf templates 
with HTML and CSS. This project is ideal for showcasing backend and frontend skills to potential employers.

Features:
- Add, edit, delete, and view employee details
- MySQL database integration with Spring Data JPA
- Thymeleaf templates for dynamic HTML pages
- User-friendly interface styled with HTML & CSS
- Ready-to-run Spring Boot application

Tech Stack:
- Java 17+
- Spring Boot 4.x
- Spring Data JPA
- MySQL
- Thymeleaf
- HTML & CSS
- Maven
